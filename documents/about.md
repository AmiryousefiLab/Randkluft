# Introduction

**Chloroplot** is a comprehensive tool for diverse visualization of organellar genomes (plastid and mitochondrial). It extract information from [GenBank](https://www.ncbi.nlm.nih.gov/genbank/) format files to generate the high-quality circular genome map. Besides the basic features, Chloroplot also provides options to visualize other important features, for example, reversed SSC region, codon usage per gene, etc.
